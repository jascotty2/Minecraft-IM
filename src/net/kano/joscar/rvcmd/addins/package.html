Provides a set of classes implementing the structure of the three types of rendezvouses sent to use AOL Instant Messenger's "add-ins" feature. An "add-in" is a game or other external program (like Quake II and MS Hearts) for which a user can use AIM to invite buddies to play or join. @see net.kano.joscar.snaccmd.CapabilityBlock#BLOCK_ADDINS
